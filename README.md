# SD albos system

## git手順

作業を始める前に

`$ git pull origin master`

～～作業をする～～


```
$ git pull origin master

$ git add .

$ git commit -m "なにをしたのかをかく"

$ git push origin master
```

- git add (作業したのをステージングする)
- git commit (ステージングされたものをローカルにあげる)
- git push (ローカルにるものをリモートのgithubにあげる)
- git pull (最新版を自分のpcにダウンロードする)
